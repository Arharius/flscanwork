━━━ КАК РАБОТАЕТ БОТ ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Пользователь пишет /start →
  Шаг 1: Нажимает кнопку "Подписаться на @dnorca"
          → Нажимает "✅ Я подписался — проверить"
  Шаг 2: Вводит секретное слово «хочу»
          → Получает ссылку на Яндекс.Диск


━━━ ЗАПУСК ЗА 3 ШАГА ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ШАГ 1 — Создайте бота в Telegram:
  1. Откройте @BotFather → /newbot
  2. Введите имя и username бота
  3. Скопируйте токен: 1234567890:ABCdef...
  4. Добавьте бота администратором канала @dnorca
     (Канал → Управление → Администраторы → добавить бота)

ШАГ 2 — Задеплойте на Render.com (БЕСПЛАТНО):
  1. render.com → New → Web Service → Connect GitHub
  2. Выберите этот репозиторий, папка: sub_bot
     Build: pip install -r requirements.txt
     Start: python bot.py
  3. Environment Variables:
       BOT_TOKEN  = ваш токен от @BotFather   ← ЕДИНСТВЕННОЕ ЧТО НУЖНО
       CHANNEL_ID = @dnorca                    (уже настроено)
       SECRET_WORD= хочу                       (уже настроено)
       DISK_LINK  = https://disk.yandex.ru/d/c6M_CbirwiK2vQ (уже настроено)
       PORT       = 10000                      (уже настроено)
  4. Deploy → готово!

ШАГ 3 — Протестируйте:
  Найдите бота в Telegram → /start → подпишитесь → введите «хочу» → ✅


━━━ ИНСТРУКЦИЯ ДЛЯ ПОЛЬЗОВАТЕЛЕЙ КАНАЛА ━━━━━━━━━━━━━━━━━━━━━━━━

  Разместите в канале @dnorca:

  "Чтобы получить материалы:
   1. Напишите боту @ВАШ_БОТ_USERNAME команду /start
   2. Подпишитесь на канал (кнопка в боте)
   3. Введите секретное слово из закреплённого поста"


━━━ СМЕНА НАСТРОЕК ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Все настройки меняются через Environment Variables на Render:
  • SECRET_WORD — новое секретное слово
  • DISK_LINK   — новая ссылка на файл
  • CHANNEL_ID  — другой канал
  После изменения Render перезапускает бота автоматически.


━━━ ТЕХНИЧЕСКИЕ ДЕТАЛИ ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  • Хостинг: Render.com (free tier)
  • Режим: polling + HTTP health-check сервер
  • БД: SQLite (хранит шаг каждого пользователя)
  • Нагрузка: 200-500 одновременных пользователей ✓
  • python-telegram-bot 20.7, asyncio
