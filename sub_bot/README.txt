═══ КАК РАБОТАЕТ БОТ ═══════════════════════════════════════════════

Пользователь пишет /start →
  Шаг 1: Подписаться на @dnorca (кнопка + проверка)
  Шаг 2: Ввести секретное слово «хочу»
  → Получает ссылку: https://disk.yandex.ru/d/c6M_CbirwiK2vQ

═══ ЧТО НУЖНО ДЛЯ ЗАПУСКА ══════════════════════════════════════════

Шаг 1 — Создайте бота в Telegram:
  1. Откройте @BotFather в Telegram
  2. Напишите /newbot
  3. Введите имя бота (например: Материалы dnorca)
  4. Введите username (например: dnorca_materials_bot)
  5. Скопируйте токен вида: 1234567890:ABCdef...

Шаг 2 — Добавьте бота администратором канала @dnorca:
  1. Откройте канал @dnorca → Управление каналом
  2. Администраторы → Добавить администратора
  3. Найдите вашего бота по username
  4. Дайте права: минимум «Добавление участников» (для проверки подписки)

Шаг 3 — Разверните на Render.com (бесплатно):
  1. Зарегистрируйтесь на render.com
  2. New → Web Service → Connect GitHub repository
  3. Выберите репозиторий с ботом
  4. Настройки:
       - Name: dnorca-sub-bot (любое)
       - Runtime: Python 3
       - Build Command: pip install -r requirements.txt
       - Start Command: python bot.py
  5. Environment Variables — добавьте:
       BOT_TOKEN     = ваш токен от @BotFather
       CHANNEL_ID    = @dnorca
       SECRET_WORD   = хочу
       DISK_LINK     = https://disk.yandex.ru/d/c6M_CbirwiK2vQ
       WEBHOOK_URL   = https://ВАШ-ДОМЕН.onrender.com
  6. Нажмите Deploy

Шаг 4 — Проверьте бота:
  1. Найдите бота в Telegram по username
  2. Напишите /start
  3. Подпишитесь на @dnorca → нажмите "Я подписался"
  4. Введите слово «хочу»
  5. Получите ссылку ✅

═══ ИЗМЕНЕНИЕ НАСТРОЕК ═════════════════════════════════════════════

Всё управляется через Environment Variables на Render:
  SECRET_WORD  — смените секретное слово
  DISK_LINK    — замените ссылку на новый файл
  CHANNEL_ID   — подключите другой канал

После изменения переменных Render перезапустит бота автоматически.
