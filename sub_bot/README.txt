━━━ КАК РАБОТАЕТ БОТ ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Пользователь пишет боту в VK →
  Шаг 1: Кнопка «Подписаться на @dnorca» (Telegram) + «Я подписался»
  Шаг 2: Ввод секретного слова «хочу»
          → Получает ссылку на Яндекс.Диск


━━━ ЗАПУСК ЗА 3 ШАГА ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ШАГ 1 — Подготовьте VK-сообщество:
  1. Зайдите в сообщество ВКонтакте → Управление
  2. API → Ключи доступа → Создать ключ
     • Поставьте галочку: Сообщения сообщества
     • Поставьте галочку: Доступ к API в любое время
  3. Скопируйте ключ доступа (VK_TOKEN)
  4. Узнайте ID сообщества: vk.com/club[ЧИСЛО] → это и есть VK_GROUP_ID
  5. Управление → Сообщения → Включить
  6. Управление → Работа с API → Long Poll API → Включить
     Версия API: 5.131, событие: Входящие сообщения ✓

ШАГ 2 — Задеплойте на бесплатный хостинг:

  Вариант А: PythonAnywhere (pythonanywhere.com) — ПРОЩЕ ВСЕГО
  1. Зарегистрируйтесь (бесплатно)
  2. Files → Upload files: загрузите vk_bot.py и requirements.txt
  3. Consoles → Bash:
       pip install vk-api python-dotenv
  4. В том же Bash задайте переменные:
       export VK_TOKEN="ваш_токен"
       export VK_GROUP_ID="123456789"
       export TG_CHANNEL="dnorca"
       export SECRET_WORD="хочу"
       export DISK_LINK="https://disk.yandex.ru/d/c6M_CbirwiK2vQ"
       python vk_bot.py
  5. Чтобы бот работал постоянно: Tasks → Always-on task → python vk_bot.py

  Вариант Б: Render.com (render.com)
  1. Зарегистрируйтесь, New → Web Service → GitHub
  2. Загрузите файлы в репозиторий GitHub
  3. Build: pip install -r requirements.txt | Start: python vk_bot.py
  4. Environment Variables: VK_TOKEN, VK_GROUP_ID, TG_CHANNEL, SECRET_WORD, DISK_LINK

ШАГ 3 — Протестируйте:
  Напишите любое сообщение боту в VK → «Я подписался» → «хочу» → ✅


━━━ ИНСТРУКЦИЯ ДЛЯ ПОЛЬЗОВАТЕЛЕЙ ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Опубликуйте в вашем VK-сообществе:

  «Хотите получить материалы?
   Напишите нашему боту: [ссылка на сообщения сообщества]
   Подпишитесь на Telegram-канал t.me/dnorca
   и введите секретное слово 😉»


━━━ СМЕНА НАСТРОЕК ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Меняйте через переменные окружения на хостинге:
  • SECRET_WORD — новое секретное слово
  • DISK_LINK   — новая ссылка на материалы
  • TG_CHANNEL  — другой Telegram-канал
