# Viber-бот для интернет-магазина

Бот для автоматизации заказов и уведомлений в Viber. Позволяет принимать заказы, отправлять статусы доставки и управлять базой товаров.

## Требования

- Python 3.8+
- Аккаунт Viber с созданным ботом (полученный `AUTH_TOKEN`)
- База данных (поддерживается SQLite/PostgreSQL)

## Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd <project-directory>
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Настройка (.env)

Создайте файл `.env` на основе `.env.example`:

```env
VIBER_AUTH_TOKEN=your_viber_bot_token_here
DATABASE_URL=sqlite:///shop.db
ADMIN_ID=your_admin_viber_id
```

- `VIBER_AUTH_TOKEN`: Токен бота из панели Viber
- `DATABASE_URL`: Строка подключения к БД
- `ADMIN_ID`: ID администратора для уведомлений

## Запуск

Активируйте бота:
```bash
python bot.py
```

Бот запустится и будет ожидать входящих сообщений.

## Тестирование

1. Добавьте бота в контакты Viber по QR-коду/ссылке
2. Отправьте команду `/start` для инициализации
3. Проверьте основные сценарии:
   - Просмотр каталога товаров
   - Оформление заказа
   - Проверка статуса доставки

Для тестирования уведомлений администратора используйте ID из `.env`.
